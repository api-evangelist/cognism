# Cognism (cognism)

Cognism is a B2B sales intelligence platform providing verified emails, mobile numbers (Diamond Data), firmographic data, intent signals, and enrichment. Cognism exposes APIs for enrichment and integration but they are gated and not publicly self-serve; access is granted to enterprise customers via partnerships and integrations.

## APIs

### Cognism Enrichment API
REST API for enriching CRM records with contact and company data, including verified emails, phone numbers, job titles, and firmographics. Available to enterprise Cognism customers; endpoints and quotas are configured per contract.
- **Base URL:** `https://api.cognism.com`
- **Docs:** https://www.cognism.com/api
